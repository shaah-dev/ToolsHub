# ⚡ ToolsHub

A beautiful, fast, and fully offline file converter web app built for everyone — Convert images, documents, and more instantly in your browser.

## ✨ Features

- **12 Powerful Tools** — Image converters, PDF tools, utilities & more
- **Preview Before Download** — See result first, then download
- **Fully Responsive** — Works perfectly on Mobile, Tablet & Desktop
- **100% Client-Side** — No data leaves your browser (Privacy First)
- **Modern & Catchy UI** — Animated, vibrant design

### Available Tools

- WebP to PNG
- PNG to JPG
- JPG to PNG
- Image to PDF
- Merge Images to PDF
- Word to PDF
- Image Compressor
- Resize Image
- QR Code Generator
- Password Generator
- Stylish Text Generator
- Base64 Converter

## 🚀 How to Use

1. Open `(https://tools-hubs.vercel.app/)` in any browser
2. Start converting instantly!

## 🛠 Tech Stack

- HTML5 + CSS3 + Vanilla JavaScript
- Tailwind CSS
- jsPDF, Mammoth.js, QRCode.js
